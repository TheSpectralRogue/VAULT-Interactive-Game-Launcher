# VAULT-Interactive-Game-Launcher
==<ABOUT THIS APPLICATION>==

This is a relatively simple launcher that helps you launch your games faster without the slow bootup of other launchers like Steam, EA, and others. You can add any program on your PC to be instantly launched, and there is a search bar and a favorite option available. I have also added a feature that tracks the amount of time you spend on applications you have added to the launcher. I also have a really cool animation on start-up, but that's the only delay you'll face when booting the launcher.

==<HOW TO USE>==
1. Launch the VAULT (Virtual Archive for User-Launchable Titles) Interactive Game Launcher.
2. Take notice of the really awesome and carefully crafted start-up animation.
3. Press the button in the top-right that says "Add Game"
4. Start by adding the executable path. You will notice that the Launcher writes the name for you, which you can change if you want.
5. (Optional) Add the path to a cover image for the game.
6. Have fun with your games and track your time on them!

==<MORE INFO>==
Data Storage: Game library and play time data are saved to Documents\VAULT\vault_library.json. You can back this up or transfer it to another PC.
Removing Games: Click the trash icon on any game card to remove it from the launcher. This does not uninstall the game from your PC.
Favorites: Click the star icon to toggle favorite status. Use the "☆ Favorites" filter to see only your starred games.
Recently Played: The "🕐 Recent" filter sorts games by last played date, most recent first.
Play Time Tracking: Time is only recorded if you play for more than 30 seconds. This prevents accidental clicks from inflating your stats.

==<TROUBLESHOOTING>==
Game won't launch? Make sure the executable path still points to a valid .exe file. If you moved or renamed the game folder, you'll need to update the path.
Cover image not showing? Supported formats: .png, .jpg, .jpeg, .gif, .bmp. Make sure the file path is correct.
Lost your library? Check that Documents\VAULT\vault_library.json exists. You can restore it from a backup if needed.
Playtime not showing? The Launcher only records playtimes that exceed 30 seconds.

- No additional software required
